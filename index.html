O проекте

Проект ставит перед собой цель ознакомить человека с интересными местами Заславского канала, проект представляет из себя pdf–документацию и интерактивную карту. Благодаря которым вы узнаете скрытые места данного города которые просто так не найти.

Важно дать пользователю представление о том, когда и каким образом были сформированы эти места. Например, историю Заславского канала и его значение для местной культуры. С течением времени вокруг канала возникали различные социокультурные пространства, которые формировали своеобразный облик города и начали скрывать канал.

Благодаря своему удобному расположению Заславский канал используется для плавания на байдарках, ловли рыбы и использования его на различных народных праздниках таких как Купалье. Маршрут по данным объектам отлично закольцовывает Заславль с его уникальной тысячелетней историей которая так-же будет интересна для изучения
